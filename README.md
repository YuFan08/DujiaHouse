# 杜家小屋 | 资深教学设计师个人空间 🏠

这是一个基于 **VitePress** 搭建的现代静态网站，集个人博客、教学设计作品展示与好物推荐于一体。

## 🌟 网站核心板块
- **首页**: 诗意化的个人品牌展示。
- **我的博客 (`/posts`)**: 记录教学心得与获奖案例分析。
- **一对一写稿 (`/about`)**: 资深教学设计师的业务介绍与服务承诺。
- **杜家好物 (`/shop`)**: 精选教育资源与工具推荐。

## 🚀 快速开始

### 1. 本地开发
在本地运行网站进行预览：
```bash
npm run docs:dev
```
访问地址：`http://localhost:5173`

### 2. 添加新博文
在 `posts/` 目录下新建 `.md` 文件，并在 `posts/index.md` 和 `.vitepress/config.mts` 中添加链接。

### 3. 一键发布
修改完成后，运行以下命令即可同步到线上域名 `www.dujia.lol`：
```bash
npm run pub
```

## 🛠️ 技术架构

- **部署**: [Vercel](https://vercel.com/)
- **域名**: [dujia.lol](https://dujia.lol)
